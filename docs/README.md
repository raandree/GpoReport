## Quick Start

The main scripts are located in the parent directory:
- `Search-GPOSettings.ps1` - For PowerShell-exported GPO XMLs
- `Search-GPMCReports.ps1` - For GPMC-generated XML reports
- `Test-GPMCSearch.Tests.ps1` - Pester test suite for validation
